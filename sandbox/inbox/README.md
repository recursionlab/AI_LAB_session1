# Inbox

Drop raw captures here.

This is intake only.

Never promote raw dumps to canon.

### Dump template (paste this header)

```
Title:
Source (link/file):
Date:
Tags:

Raw dump:
- 

Notes:
- 
```

### Worked example (minimal)

```
Title: Kleiner (2024) structural turn — rough notes
Source (link/file): DOI:10.1016/j.concog.2024.103653
Date: 2026-02-16
Tags: consciousness, structure, isomorphism

Raw dump:
- Key distinction: structural vs structuralist (automorphisms).
- ISO claim is strong; not automatic explanation.
- Methodology matters: phenomenal structure depends on construction rules.

Notes:
- Candidate action: rename our “isomorphism” docs to “mappings”.
```

### Extract prompt (5 minutes)

Take one dump and output only:

1. **Concepts** (new nouns we should define)
2. **Operators** (new symbols or transformations)
3. **Claims** (assertions we rely on)
4. **Tests / falsifiers** (what would prove it wrong)
5. **Issues / tensions** (contradictions, missing definitions)

Then route each bullet:

* **Promote** (turn into a canon page)
* **Park** (leave in Draft specs)
* **Kill** (delete)

Routing home: [Ops + Triage](../ops-+-triage/).
