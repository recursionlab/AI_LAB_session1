# Worklog

Use this page as the **external memory** for speedruns.

### How to use

* Append one short entry whenever we:
  * resolve a tension
  * change a definition/operator/equation
  * create/move/delete a page
  * decide “Promote / Park / Kill” for a chunk of material
* Keep entries **tiny**.

### Entry template

Copy/paste:

```
Date:
Focus:
Changed:
Decisions:
Next:
Links:
```

### Log

```
Date: 2026-02-16
Focus: Speedrun continuity scaffolding
Changed: Added Living System Brief “Speedrun mode” + created Worklog + upgraded tension ledger workflow.
Decisions: GitBook is the persistence layer; resume via Brief → latest Worklog → Next action.
Next: Fill the state vector in Living System Brief (current focus / next action / primary blocker).
Links: Living System Brief, Ops + Triage, Worklog
```
