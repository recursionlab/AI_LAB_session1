# Draft specs

Integrated drafts live here.

They can be long. They can be inconsistent.

### Promotion rule

When a section stabilizes, split it into canon pages.

Keep the draft as the “working source” until the split is complete.

### Promotion threshold checklist (use this to decide “stable”)

A section is ready to promote when it has:

1. **One-sentence definition** (what it is, not what it gestures at).
2. **Scope** (what it is not / what it does not cover).
3. **Dependencies** (links to required primitives/operators).
4. **Assumptions** (explicit list).
5. **Mechanism** (a rule, operator law, or equation stub).
6. **Failure mode / falsifier** (what would break it).
7. **Landing zone** (which doc type it becomes: Foundations / Operators / Equations / Interpretations).
8. **Cross-links** (at least 2 relevant pages).

If any item is missing, keep it in Draft specs.
